# svelte-fundamentals
Example code on the the training Svelte Fundamentals, by Peter Kassenaar

## Useful Svelte resources on the internet

- Official Svelte Documentation – The primary reference for Svelte fundamentals, reactivity, components, stores, and lifecycle hooks.
  https://svelte.dev/docs
- Official SvelteKit Documentation – Guides for routing, data loading (load), form actions, adapters, and building full-stack Svelte applications.
  https://svelte.dev/docs/kit
- Svelte Society – Curated links to articles, talks, tooling, and community projects within the Svelte ecosystem.
  https://www.sveltesociety.dev
- JoyOfCode - YouTube channel with tutorials, screencasts, and explanations of Svelte concepts. Look at the playlist for the latest Svelte tutorials!
  https://www.youtube.com/@JoyofCodeDev/playlists
- SvelteMastery - YouTube channel dedicated to Svelte ecosystem topics, Interviews with Svelte maintainers and ecosystem authors, tutorials and architectural discussions. https://www.youtube.com/@SvelteMastery
